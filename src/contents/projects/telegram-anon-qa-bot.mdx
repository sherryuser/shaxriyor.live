## Getting Started in Minutes ‚è∞

Setting up the Telegram Anonymous Q&A Bot is a breeze. Follow these simple steps:

1. **Clone the Repository:**
```
git clone https://github.com/sherryuser/telegram-anon-qa-bot.git
   ```

2. **Navigate to the Project:**
```
cd telegram-anon-qa-bot
   ```

3. **Install Dependencies:**
```
pip3 install telebot
   ```

4. **Update API Key:**
Replace `YOUR_TELEGRAM_BOT_API_KEY` in the bot script with your actual Telegram Bot API key.

5. **Run the Bot:**
```
python main.py
   ```

Now, your Telegram Anonymous Q&A Bot is up and running, ready to deliver an anonymous Q&A experience like no other!
